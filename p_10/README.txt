Para compilar el programa se pondra:
javac Converter.java Main.java

Y para ejecutarlo:
java Main <tus_pruebas.txt> <tu_output.txt>
Ejemplo:
java Main test_cases.txt output.txt

El segundo parametro solamente indica como
se llamara el archivo que se generara donde
estar el output deseado.
